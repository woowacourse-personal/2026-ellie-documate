# DocuMate

> 개발 문서의 단어만 번역하지 않고, 기술적인 맥락까지 이해합니다.

DocuMate는 영어로 된 개발 문서와 기술 아티클을 개발 용어와 기술적인 맥락에 맞게 번역하고 해설하는 Chrome 확장 프로그램입니다.

구글 번역이나 일반 번역 확장 프로그램으로 개발 문서를 번역하면 개발 용어가 일상적인 의미로 번역되거나, 문장의 기술적인 맥락과 다른 결과가 나오는 경우가 있습니다.

DocuMate는 개발 문서의 용어와 앞뒤 맥락을 고려해 자연스러운 한국어 번역을 제공합니다. 번역만으로 이해하기 어려운 내용은 개념 해설을 통해 확인하고, 궁금한 점이 남아 있다면 후속 질문을 이어갈 수 있습니다.

## 해결하고자 하는 문제

영어 개발 문서를 읽을 때 번역 도구를 사용해도 내용을 정확하게 이해하기 어려운 경우가 있습니다.

개발 용어는 일반적인 영어 단어와 같은 형태를 사용하면서도 기술 문맥에서는 다른 의미를 가질 수 있습니다. 기존 번역 도구가 이러한 맥락을 충분히 고려하지 않으면 용어를 어색하게 번역하거나 문장의 의도와 다른 결과를 제공하기도 합니다.

예를 들어 `state`, `scope`, `context`, `suspend`, `callback`과 같은 표현은 일반적인 사전적 의미보다 해당 기술과 문서 안에서 어떤 의미로 사용되었는지가 중요합니다.

결국 사용자는 번역 결과를 다시 해석하거나, 개발 용어를 별도로 검색하고, 여러 자료를 비교해야 합니다. 번역을 사용했음에도 문서를 이해하기 위해 또 다른 탐색 과정이 필요한 것입니다.

DocuMate는 이러한 불편을 줄이기 위해 다음에 집중합니다.

- 개발 용어를 기술 문맥에 맞게 해석합니다.
- 문장 하나만이 아니라 앞뒤 내용을 고려해 번역합니다.
- 직역만으로 이해하기 어려운 개념을 별도로 해설합니다.
- 번역과 해설 이후에도 후속 질문을 통해 이해를 이어갈 수 있도록 합니다.

## 주요 기능

### 개발 문맥을 고려한 번역

영어 개발 문서와 기술 아티클을 개발 용어와 문서의 흐름에 맞는 자연스러운 한국어로 번역합니다.

일반적인 사전적 의미를 그대로 적용하기보다, 해당 용어가 현재 문서와 기술 분야에서 어떤 의미로 사용되었는지를 고려합니다.

### 개발 개념 해설

번역된 문장만으로 이해하기 어려운 개발 용어와 기술 개념을 쉽게 풀어 설명합니다.

단순히 문장이 무엇을 뜻하는지만 알려주는 것이 아니라, 해당 개념이 왜 필요한지와 어떤 상황에서 사용되는지까지 이해할 수 있도록 돕습니다.

### 후속 질문

번역이나 해설을 확인한 뒤에도 궁금한 점이 남아 있다면 추가 질문을 입력할 수 있습니다.

현재 읽고 있는 문서와 이전 답변의 맥락을 바탕으로 설명을 이어갈 수 있어, 궁금한 내용을 별도로 검색하지 않고 단계적으로 확인할 수 있습니다.

### 필요한 부분만 드래그하여 선택

특정 문장이나 문단만 자세히 확인하고 싶다면 원하는 영역을 드래그하여 선택할 수 있습니다.

선택한 내용을 기준으로 번역과 해설을 요청하여 필요한 부분에 집중할 수 있습니다.

## 기존 번역 도구와의 차이

| 구분 | 일반 번역 도구 | DocuMate |
|---|---|---|
| 번역 기준 | 일반적인 언어 표현 | 개발 용어와 기술 문맥 |
| 개발 용어 | 사전적 의미로 번역될 수 있음 | 기술 분야에서 사용되는 의미를 고려 |
| 결과 제공 | 번역문 중심 | 번역과 개념 해설 제공 |
| 추가 탐색 | 별도의 검색이 필요할 수 있음 | 현재 내용에 관한 후속 질문 가능 |
| 사용 흐름 | 번역과 검색 도구를 오갈 수 있음 | 문서를 읽는 페이지에서 이어서 확인 |

## 이런 분에게 추천합니다

- 영어 공식 문서와 기술 아티클을 자주 읽는 개발자
- 번역 결과에서 개발 용어가 엉뚱하게 해석되어 불편했던 개발자
- 문장은 번역되었지만 기술적인 의미를 이해하기 어려웠던 개발자
- 번역 결과를 확인한 뒤 다시 개발 용어를 검색해야 했던 개발자
- 영어 개발 문서를 읽고 이해하는 부담을 줄이고 싶은 개발 학습자

## 사용 방법

1. DocuMate Chrome 확장 프로그램을 설치합니다.
2. 영어 개발 문서나 기술 아티클에 접속합니다.
3. 툴바의 DocuMate 아이콘을 눌러 실행하고 번역 결과를 확인합니다.
4. 이해하기 어려운 내용은 개념 해설을 확인합니다.
5. 궁금한 점이 남아 있다면 후속 질문을 입력합니다.
6. 특정 부분만 확인하려면 문장이나 문단을 드래그하여 선택합니다.

> 아이콘을 누른 탭에서만 동작하며, 누르기 전에는 어떤 페이지에서도 콘텐츠를 읽거나 서버를 호출하지 않습니다.
