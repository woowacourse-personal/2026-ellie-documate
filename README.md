# Documate

> 영어 개발 문서를 읽다 막힌 자리에서, 페이지를 떠나지 않고 이해하도록 돕는 크롬 확장 프로그램

일반 번역기는 개발 문서에서 코드·API 이름·용어를 엉뚱하게 옮기고 맥락을 놓쳐 오히려 방해가 된다. Documate는 코드·API 이름·식별자는 원문 그대로 두고, 맥락을 살린 자연스러운 한국어로 각 문단을 옮긴다.

더 파고들고 싶은 자리에선 버튼 한 번으로 해설을 그 자리에서 펼친다. 일반론이 아니라 이 문서 맥락 안에서 "왜/언제 쓰는지"를, 코드에는 "무엇을 하는지"를 풀어준다.

## 기능

- **문단 번역** — 각 영어 문단 아래 한국어. 식별자·코드는 원문 유지
- **개념 해설** — 문장에 버튼 → 인라인 펼침, 문서 맥락 기반(왜/언제)
- **코드 해설** — 코드 블록엔 번역 대신 "무엇을 하는지" 해설
- **후속 질문** — 해설 안에서 탭 이동 없이 이어 질문
- **드래그 선택** — 아무 텍스트나 드래그하면 그 자리에서 번역·해설

## 기술 스택

Chrome MV3 · Vite + CRXJS · 순수 DOM(Shadow DOM) · @mozilla/readability · Vercel Functions · Gemini (flash-lite, 번역·해설)
