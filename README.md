# Documate

> 영어 개발 문서를 읽다 막힌 자리에서, 페이지를 떠나지 않고 이해하도록 돕는 크롬 확장

영어 개발 문서를 읽을 때 **문단마다 한국어 번역을 항상** 보여주고, **개념 해설은 버튼 한 번으로** 그 자리에서 펼쳐 본다. 번역이 아니라 "왜/언제 쓰는지"를 풀어주는 게 핵심.

## 기능 (MVP)

- **문단 번역** — 각 영어 문단 아래 한국어 번역
- **개념 해설** — 문단별 버튼 → 인라인 펼침 (무엇/왜/언제)
- **후속 질문** — 해설 안에서 탭 이동 없이 이어 질문
- **드래그 폴백** — 문단 인식 실패 페이지는 드래그로 번역+해설

## 동작

확장 아이콘을 클릭한 **그 탭에서만** 켜진다. 누르기 전엔 아무 동작 안 함.

## 기술 스택

Chrome MV3 · Vite + CRXJS · Preact(Shadow DOM) · @mozilla/readability · Vercel Functions · Gemini
