# DocuMate 개인정보 처리방침 / Privacy Policy

*최종 수정: 2026-07-19*

DocuMate("본 확장")는 영어 개발 문서를 번역·해설해 이해를 돕는 Chrome 확장 프로그램입니다. 본 문서는 본 확장이 어떤 데이터를 다루고, 어디로 보내며, 무엇을 보관하는지 설명합니다.

---

## 1. 요약 (핵심만)

- 본 확장은 **아이콘을 누른 탭에서만** 동작합니다(`activeTab` 권한). 누르기 전에는 어떤 페이지에서도 내용을 읽거나 전송하지 않습니다.
- 번역·해설을 위해, 사용자가 활성화한 페이지의 **본문 텍스트** 또는 **드래그로 선택한 텍스트**를 처리 서버(프록시)를 거쳐 **Google Gemini**로 보냅니다.
- **계정·로그인·비밀번호·결제정보를 수집하지 않습니다.** 방문 기록을 추적하지 않고, 광고·분석 트래커를 심지 않습니다.
- 반복 번역 비용을 줄이기 위해 **번역 결과**를 캐시합니다. 단 **드래그로 선택한 텍스트는 공용 캐시에 저장하지 않습니다**(사내·비공개 문서일 수 있으므로).

---

## 2. 수집·처리하는 데이터

본 확장이 기능 제공을 위해 처리하는 데이터는 다음과 같습니다.

| 데이터 | 언제 | 목적 | 어디로 |
|---|---|---|---|
| 페이지 본문 텍스트(문단·제목·콜아웃) | 사용자가 아이콘을 눌러 활성화한 페이지에서 | 번역 | 프록시 → Google Gemini |
| 드래그로 선택한 텍스트 | 사용자가 텍스트를 선택하고 번역/해설을 요청할 때 | 번역·해설 | 프록시 → Google Gemini |
| 해설 대상 문장/코드 + 문서 맥락(문서 제목·앞 문단) | 사용자가 해설 버튼을 누를 때 | 개념·코드 해설 | 프록시 → Google Gemini |
| 후속 질문과 대화 내역 | 사용자가 해설에 후속 질문을 할 때 | 대화형 해설 | 프록시 → Google Gemini |
| IP 주소 | 모든 프록시 요청 시 | 남용 방지(레이트리밋) | 프록시(임시 카운터) |

**본 확장이 수집하지 않는 것:** 이름·이메일·전화번호 등 신원정보, 계정·비밀번호, 결제·금융정보, 전체 방문 기록, 위치정보, 마케팅·분석용 추적 식별자.

---

## 3. 제3자 서비스

번역·해설 기능은 다음 외부 서비스를 사용합니다. 위 데이터가 해당 서비스로 전송됩니다.

- **Google Gemini API** — 전송한 텍스트를 번역·해설로 처리합니다. Google의 API 데이터 사용 정책이 적용됩니다.
- **Vercel** — 처리 서버(프록시)를 호스팅합니다.
- **Upstash (Redis)** — 공용 번역 캐시와 레이트리밋 카운터를 저장합니다.

본 확장은 이 외의 제3자에게 데이터를 판매하거나 공유하지 않습니다.

---

## 4. 데이터 보관

- **공용 번역 캐시** — 같은 문장을 반복 번역하지 않도록 **번역된 문단 텍스트**를 서버 캐시에 저장합니다. **드래그로 선택한 텍스트는 저장하지 않습니다.**
- **로컬 캐시** — 번역 결과는 사용자 브라우저의 `chrome.storage.local`에도 저장되어, 확장을 제거하면 함께 삭제됩니다.
- **IP 주소** — 레이트리밋 목적의 임시 카운터로만 쓰이며, 시간 창이 지나면 자동 만료됩니다. 별도의 사용자 프로필로 축적하지 않습니다.

---

## 5. 권한 사용 이유

- **`activeTab`** — 사용자가 아이콘을 누른 그 탭의 내용에만 접근하기 위함입니다. 상시 전체 웹 접근 권한(`<all_urls>`)을 요구하지 않습니다.
- **`storage`** — 번역 결과를 로컬에 캐시해 재번역을 줄이기 위함입니다.
- **호스트 접근(프록시 도메인)** — 번역·해설 요청을 본 확장의 처리 서버로 보내기 위함입니다.

---

## 6. 사용자 선택권

- 아이콘을 누르지 않으면 본 확장은 아무 데이터도 처리하지 않습니다.
- 민감한(사내·비공개) 텍스트는 **드래그 선택 방식**으로 요청하면 공용 캐시에 저장되지 않습니다.
- Chrome의 확장 관리 화면에서 언제든 본 확장을 제거할 수 있으며, 제거 시 로컬 캐시도 삭제됩니다.
